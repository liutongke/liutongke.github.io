Quit Packet 🔗

MySQL客户端需要断开与MySQL服务端连接时，需要向服务端发送一个Quit Packet指令。这个包比较简单，关闭当前连接（客户端退出），无参数。前面的四个字节是包头，其中前三位代表packet数据长度，第四位是包序列号。以下是Quit Packet的四字节后的内容。只有一个包头标识数据。

相对包内容的位置 长度（字节） 名称 描述
0 1 包头标识 0x01 代表这是一个Quit 包
