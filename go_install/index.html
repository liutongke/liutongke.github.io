Linux安装Go

2024年01月09日 · 370字 · 1分钟 · 阅读量次

在 Linux 上安装 Go 通常包括下载 Go 二进制分发版本并将其添加到系统的 PATH 中。以下是一般的步骤：

  1. 下载 Go 二进制分发版本：

    • 访问 官方 Go 下载页面。

    • 选择适用于你 Linux 系统的版本（通常是 Linux 和 64-bit）。你可以在下载页面上找到最新的版本。

    • 复制下载链接，然后使用 wget 或 curl 命令下载：

      wget https://golang.org/dl/goX.X.X.linux-amd64.tar.gz

      或

      curl -O https://golang.org/dl/goX.X.X.linux-amd64.tar.gz

  2. 解压缩下载的文件：

    • 使用以下命令解压缩下载的 Go 压缩文件：

      sudo tar -C /usr/local -xzf goX.X.X.linux-amd64.tar.gz

  3. 配置环境变量：

    • 将 Go 的 bin 目录添加到你的 PATH 中。可以在你的 shell 配置文件（比如 ~/.bashrc、~/.zshrc）中添加以下行：

      export PATH=$PATH:/usr/local/go/bin

      或者使用 echo 命令将上述行添加到文件中：

      echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc

      然后运行以下命令应用更改：

      source ~/.bashrc   # 或者 source ~/.zshrc，根据你的终端配置文件

  4. 验证安装：

    • 打开新的终端窗口，并运行以下命令：

      go version

    • 如果 Go 安装成功，将显示 Go 的版本信息。

现在，已经在 Linux 系统上成功安装了 Go。确保根据实际情况替换命令中的 goX.X.X 为你下载的 Go 版本。
